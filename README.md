# PrimeRobo-by-Prime
An AI that learns from the user and can use pre-built knowledge from Wikipedia and browsers

How to Open the App

1. Download the zip file
2. Extract the Zip file
3. For Windows: Run the .bat file; for Mac: Run the .sh file
4. Enjoy the Chatbot

You may train the AI and show it off to your friends or your family (Please do not publish/sell your copy of this app)
